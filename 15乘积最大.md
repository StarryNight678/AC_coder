
# 乘积最大

题目描述
									
有一个整数n，将n分解成若干个不同自然数之和，问如何分解能使这些数的乘积最大，输出这个乘积m

输入
一个整数
样例输入
15

输出
一个整数
样例输出
144

## 说明


 c[i][j] = Math.max(c[i][j], c[i - j][k] * j);

 i表当前的和.当前共j个数.

 若采用j.

 和变成i-j. 需要考虑j-1 个数  (j-1个数和为i-j)
 和j相乘的最大值.

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
//        Scanner sc = new Scanner(System.in);
//        int N = sc.nextInt();
        int N = 15;
        //int [] w;
        int[][] c = new int[N + 1][N + 1];
        for (int i = 0; i <= N; i++)
            c[0][i] = 1;
        for (int i = 1; i <= N; i++) {
            for (int j = 1; j <= i; j++) {
                for (int k = 1; k < j; k++) {
                    //原来和为i,共j个数
                    //考虑选择j之后. 和为i-j  共j-1 个数.
                    c[i][j] = Math.max(c[i][j], c[i - j][k] * j);
                }
            }
        }
        for (int i = 0; i <= N; i++) {
            for (int j = 0; j <= N; j++) {
                System.out.print(String.format("%3d ", c[i][j]));
            }
            System.out.println();
        }
        int maxNum = c[N][0];
        for (int i = 1; i <= N; i++) {
            maxNum = Math.max(maxNum, c[N][i]);
        }
        System.out.println("------------------------");
        System.out.println("结果:" + maxNum);
        int tmp = N;
        int[] flag = new int[N + 1];

        while (tmp > 1) {
            int tmpMax = c[tmp][0];
            int tmpMaxi = 0;
            for (int i = 1; i <= N; i++) {
                if (tmpMax < c[tmp][i]) {
                    tmpMax = c[tmp][i];
                    tmpMaxi = i;
                }
            }
            flag[tmpMaxi] = 1;
            tmp -= tmpMaxi;
        }
        System.out.println("因子如下:");
        for (int i = 1; i <= N; i++) {
            if (1 == flag[i])
                System.out.print(i + " ");
        }
    }
}
```
