# 代码面试指南


## 去掉字符串中连续出现的K个0

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        String str = "A0000B000C000";
        int k = 3;
        char[] ch = str.toCharArray();
        int len = ch.length;
        System.out.println("before:\n" + str);
        int count = 0;
        int start = -1;
        for (int i = 0; i < len; i++) {
            if (ch[i] == '0') {
                count++;
                if (-1 == start) {
                    start = i;
                }
            } else {
                if (count == k) {
                    start = -1;
                    count = 0;
                } else {
                    for (int j = 0; j < count; j++) {
                        System.out.print("0");
                    }
                    start = -1;
                    count = 0;
                }
                System.out.print(ch[i]);
            }
        }
    }
}
```
